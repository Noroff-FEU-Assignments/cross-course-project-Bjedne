Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

I would start by advicing them to look up well-known third-party security applications. Help them look into a trustworthy application that could help protect against web security attacks through implementations of sanitization of data to avoid HTMLs with malicious script to execute. I would also advise the client to incorperate several security measures against the most popular and common types of attacks recently to increase overall security. In the end I would advise them to include the whole team working on their product to make sure that everyone are on the same page from creating, updating and testing how secure the measures are and test them out against mock attacks done by a trusted professional third-party to see how good it is and better learn how to handle a breach better.
